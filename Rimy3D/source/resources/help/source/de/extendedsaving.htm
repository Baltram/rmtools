Extended Saving

Extended Saving ist ein Plugin für das 3D-Programm GMax, das den Import von .3db-Dateien und das Speichern von zusätzlichen Informationen in .gmax-Dateien ermöglicht.
Rimy3D kann nur solche .gmax-Dateien öffnen, die mit Extended Saving gespeichert wurden.

Die Installation erfolgt über den Menüpunkt 'Extras' in der Menüleiste von Rimy3D. Für den Fall, dass auf Ihrem System .gmax-Dateien nicht automatisch mit GMax geöffnet werden, müssen Sie manuell den Installationsort von gmax.exe angeben, wenn Windows nachfragt, mit welchem Programm die Datei installExtendedSaving.gmax geöffnet werden soll.

Nach der Installation steht in GMax ein zusätzlicher Menüpunkt 'Extended Saving' zur Verfügung, über den die Speicherung der zusätzlichen Informationen ein- und ausgeschaltet werden kann. Von dort aus ist auch der Import von .3db-Dateien möglich.
