Blender 3db Import/Export Scripts

Dieses Plugin für Blender 2.63 und neuer ermöglicht den Im- und Export von .3db-Dateien.

Die Installation erfolgt über den Menüpunkt 'Extras' in der Menüleiste von Rimy3D. Wählen Sie dazu das Installationsverzeichnis von Blender aus. Nachdem die benötigten Dateien automatisch kopiert wurden, kann das Plugin in Blender unter 'File' > 'User Preferences...' > 'Addons' > 'Import-Export' > 'Baltram's 3D format' aktiviert werden. Nach dem Setzen des Häkchens sollten die vorgenommenen Einstellungen per Klick auf 'Save User Settings' gespeichert werden.

Nach der Installation können .3db-Dateien über 'File' > 'Import' und 'File' > 'Export' geladen und gespeichert werden. Für den Export von Vertex Colors sollten die betreffenden Meshes jeweils einen Vertex Color-Kanal mit dem Namen 'rgb' und einen mit dem Namen 'alpha' besitzen.
