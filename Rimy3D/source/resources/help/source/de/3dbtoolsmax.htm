3db Tools für 3ds Max

Dieses Plugin für 3ds Max (alle Versionen) ermöglicht den Im- und Export von .3db-Dateien.

Die Installation erfolgt über den Menüpunkt 'Extras' in der Menüleiste von Rimy3D. Wählen Sie dazu das Installationsverzeichnis von 3ds Max aus.

Nach der Installation steht in 3ds Max ein neuer Menüpunkt '3db' zur Verfügung. Alternativ zu diesem Menüpunkt kann das Interface unter 'MaxScript' > '3db Tools' im Utilities Tab (Hammer-Symbol) verwendet werden. Dort lässt sich auch das '3db'-Menü entfernen.
