Blender 3db Import/Export Scripts

This add-on enables Blender 2.91 and newer to import and export .3db files.

It is made available from the 'Tools' menu in Rimy3D and installed using Blender's preferences menu.

Once installed, you can open and save .3db files through 'File' > 'Import' and 'File' > 'Export'.
When exporting vertex colors, the meshes should have one vertex color attribute with the name 'rgb' and one with the name 'alpha'.
