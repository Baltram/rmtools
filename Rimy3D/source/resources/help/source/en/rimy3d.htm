Rimy3D help

Welcome to the Rimy3D help!

Rimy3D is a program for converting 3D file formats, especially the game data from Gothic 3 and Risen.

For the best possible compatibility with popular 3D editors, Rimy3D offers easy to install import/export plug-ins for GMax, 3ds Max and Blender.

Overview

Rimy3D's import/export capabilities are detailed in the list of supported file formats.

Important information on the work with Risen and Gothic 3 models is available in the articles about .xact, .xcmsh, .xlmsh, .xnvmsh, ._xmac, ._xmsh and ._xcom files.
The related material files (.xshmat or ._xmat) can be edited using the built-in Genome Material-Editor.

Different ways of importing 3D files, export options, and batch conversion are explained in the section about opening and saving 3D files.
The search paths play an important role in loading materials and textures.

The controls for zooming, panning and rotating are specified in the viewport article.

Finally this help includes short instructions on how to install the plug-ins for GMax, 3ds Max and Blender.
