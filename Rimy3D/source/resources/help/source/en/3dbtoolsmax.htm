3db Tools for 3ds Max

This plug-in enables 3ds Max (all versions) to import and export .3db files.

It is installed through the 'Tools' menu in Rimy3D, you only need to specify the 3ds Max installation directory.

Once installed, there is a new menu labeled '3db' in the main menu bar of 3ds Max. Alternatively, you can open up the plug-in's interface through 'MAXScript' > '3db Tools' in the utilities tab (hammer symbol) where you can also disable the '3db' menu.
