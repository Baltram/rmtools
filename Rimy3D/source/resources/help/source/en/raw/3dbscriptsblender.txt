[SIZE=5]Blender 3db Import/Export Scripts[/SIZE]

[SIZE=2]This plug-in enables [B]Blender 2.91[/B] and newer to import and export [B].3db[/B] files.[/SIZE]

[SIZE=2]It is installed through the 'Tools' menu in [B]Rimy3D[/B]. Simply select the Blender installation directory so the plug-in files can copied over. To finish the installation, start [B]Blender[/B] and go to 'Edit' > 'Preferences...' > 'Add-ons'. Enable 'Baltram's 3D format', which is found in the 'Import-Export' category.[/SIZE]

[SIZE=2]Once installed, you can open and save [B].3db[/B] files through 'File' > 'Import' and 'File' > 'Export'.[B][/B]
When exporting vertex colors, the meshes should have one vertex color attribute with the name 'rgb' and one with the name 'alpha'.[/SIZE]
