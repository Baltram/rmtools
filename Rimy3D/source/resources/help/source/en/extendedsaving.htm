Extended Saving

Extended Saving is a plug-in that enables GMax to import .3db files and to save additional information in .gmax files.
Rimy3D can only open .gmax files that have been saved using Extended Saving.

This plug-in is installed through the 'Tools' menu in Rimy3D. In case .gmax files are not associated with the GMax application on your system, you need to manually specify the location of gmax.exe when Windows asks you how to open the file 'installExtendedSaving.gmax'.

Once installed, there is a new menu labeled 'Extended Saving' in the main menu bar of GMax. Using this menu you can enable and disable the saving of additional information in .gmax files as well as import .3db files.
