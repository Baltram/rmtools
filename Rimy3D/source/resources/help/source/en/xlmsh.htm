Gothic 3 LOD mesh (.xlmsh)

For performance reasons some Gothic 3 meshes exist at different quality levels. The file name is of the form [mesh name].xcmsh for the highest, [mesh name]_LOD1.xcmsh for the middle and [mesh name]_LOD2.xcmsh for the lowest quality level.
A .xlmsh file contains merely references to the .xcmsh files that represent the different quality levels of one mesh. These are normally two, and in rare occasions three references.

When importing an LOD mesh in Rimy3D, all referenced .xcmsh files are loaded, provided they reside in the same folder as the .xlmsh file.

When exporting an LOD mesh from Rimy3D, a reference of the form [object name].xcmsh is added to the .xlmsh file for each of the objects in the currently open scene.
